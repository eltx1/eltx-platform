# ELTX — Next.js App (v2)

## Run
```bash
npm i
echo "PRICE_URL=http://YOUR_PHP_SITE/api/price.php?provider=best" > .env.local
npm run dev
```

If you see `Module not found: './globals.css'`, make sure the file exists at `app/globals.css`. (In this zip it’s already fixed.)
